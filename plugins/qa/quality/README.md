# QA Quality Plugin

Quality assurance primitives for pre-commit checks and QA setup.

## Installation

```json
{ "plugins": { "qa/quality": true } }
```

## Skills
- `/qa-quality:pre-commit-qa` — Run standardized QA checks before committing
- `/qa-quality:qa-setup` — Verify or create standardized QA commands

## Commands
- `review.md` — Review implementation against project plan and verify completeness
