# DevOps Git Plugin

Git workflow automation for commits, merges, pushes, and reviews.

## Installation

```json
{ "plugins": { "devops/git": true } }
```

## Skills
- `/devops-git:commit` — Create conventional commits with proper formatting
- `/devops-git:review` — Review implementation against project plan

## Commands
- `merge.md` — Merge PR and cleanup branch
- `push.md` — Push to remote and wait for CI status
