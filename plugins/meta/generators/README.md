# Meta Generators Plugin

Meta-primitives for generating commands, prompts, and documentation.

## Installation

```json
{ "plugins": { "meta/generators": true } }
```

## Commands
- `create-command.md` — Generate a new command primitive
- `prompt-generator.md` — Generate agentic prompts at any level
- `create-prime.md` — Generate a repo-specific prime command
- `create-doc-sync.md` — Generate a customized doc-sync command
