# Architecture Plugin

Agent-native architecture patterns, swarm orchestration, and skill creation guides for building AI-first systems.

## Skills

- **agent-native-architecture** — Patterns and principles for designing systems where AI agents are first-class participants
- **orchestrating-swarms** — Coordinating multiple agents working together
- **create-agent-skills** — Guide to creating effective agent skills with references, templates, and workflows
