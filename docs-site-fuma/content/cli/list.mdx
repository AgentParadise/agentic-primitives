## Overview

The `list` command displays all primitives in your repository with filtering and formatting options.

## Usage

```bash
agentic-p list [OPTIONS]
```

## Options

| Option | Description | Default |
|--------|-------------|---------|
| `-t, --type <TYPE>` | Filter by type: `agent`, `command`, `skill`, `tool`, `hook` | All |
| `-c, --category <CAT>` | Filter by category | All |
| `-s, --status <STATUS>` | Filter by version status: `active`, `draft`, `deprecated` | All |
| `--format <FMT>` | Output format: `table`, `json`, `yaml`, `tree` | `table` |
| `--no-header` | Hide table header | `false` |
| `-v, --verbose` | Show additional details | `false` |

## Examples

### Basic Usage

```bash
# List all primitives
agentic-p list

# List only agents
agentic-p list --type agent

# List only hooks
agentic-p list --type hook
```

### Filtering

```bash
# Filter by category
agentic-p list --category security

# Filter by type and category
agentic-p list --type hook --category security

# Only show active versions
agentic-p list --status active
```

### Output Formats

```bash
# Tree view
agentic-p list --format tree

# JSON output (for scripting)
agentic-p list --format json

# YAML output
agentic-p list --format yaml
```

## Combining with Other Tools

### Count Primitives

```bash
# Count all primitives
agentic-p list --format json | jq '.total'

# Count by type
agentic-p list --type hook --format json | jq '.total'
```

### Script Integration

```bash
# Get primitive names
agentic-p list --format json | jq -r '.primitives[].name'

# Get paths for backup
agentic-p list --format json | jq -r '.primitives[].path'
```
