## Overview

The `inspect` command displays detailed information about a specific primitive, including metadata, versions, and content.

## Usage

```bash
agentic-p inspect <PATH> [OPTIONS]
```

## Arguments

| Argument | Description | Required |
|----------|-------------|----------|
| `<PATH>` | Primitive path (e.g., `python/python-pro`) | Yes |

## Options

| Option | Description | Default |
|--------|-------------|---------|
| `--version <N>` | Inspect specific version | Default version |
| `--show-content` | Display prompt/implementation content | `false` |
| `--show-hash` | Display BLAKE3 hash | `false` |
| `--format <FMT>` | Output format: `pretty`, `json`, `yaml` | `pretty` |
| `-v, --verbose` | Show all available information | `false` |

## Examples

### Basic Usage

```bash
# Inspect an agent
agentic-p inspect python/python-pro

# Inspect a hook
agentic-p inspect security/bash-validator

# Inspect a command
agentic-p inspect qa/review
```

### With Options

```bash
# Show specific version
agentic-p inspect python/python-pro --version 1

# Show content
agentic-p inspect python/python-pro --show-content

# Show hash
agentic-p inspect python/python-pro --show-hash

# Full verbose output
agentic-p inspect python/python-pro --verbose
```

### Output Formats

```bash
# JSON output
agentic-p inspect python/python-pro --format json

# YAML output
agentic-p inspect python/python-pro --format yaml
```
