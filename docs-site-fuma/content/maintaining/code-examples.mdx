## General Principles

<Cards>
  <Card title="Real & Runnable">
    Examples should work when copied. Test before committing.
  </Card>
  <Card title="Minimal & Focused">
    Show only what's needed. Avoid boilerplate noise.
  </Card>

    Explain non-obvious parts. Don't over-comment obvious code.
  </Card>

## CLI Examples

### Command Format

Show the command, then the expected output:

```bash
agentic-p list
```

```text
Primitives in ./primitives:

TYPE     NAME              VERSION  STATUS
agent    python-expert     v2       active
agent    code-reviewer     v1       active
command  refactor          v3       active
skill    testing-patterns  v1       draft
```

### With Flags

Group related flags logically:

```bash
# Validate with all checks
agentic-p validate \
  --strict \
  --check-hashes \
  --check-deps
```

### Error Examples

When showing errors, include the full output:

