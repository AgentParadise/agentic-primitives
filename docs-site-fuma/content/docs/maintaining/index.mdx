## Quick Reference

### File Structure

```
docs-site-fuma/
├── content/
│   └── docs/
│       ├── index.mdx         # Landing page
│       ├── concepts/         # Core concepts
│       ├── cli/              # CLI reference
│       ├── guides/           # Step-by-step tutorials
│       ├── reference/        # Technical reference
│       └── maintaining/      # This section
├── app/                      # Next.js app
├── lib/                      # Utilities
└── source.config.ts          # Content configuration
```

### Running Locally

```bash
# From lib/agentic-primitives
just docs
```

### Adding a New Page

1. Create an `.mdx` file in the appropriate directory
2. Add frontmatter with `title` and `description`
3. Add the page to the `meta.json` in that directory
4. Test locally before committing
