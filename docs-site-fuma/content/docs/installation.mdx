## Installing Dependencies

### Rust

Rust is required to build the agentic-p CLI from source.

<Tabs items={['macOS / Linux', 'Windows']}>
  <Tab value="macOS / Linux">
    ```bash
    curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
    source $HOME/.cargo/env
    rustc --version
    ```
  </Tab>
  <Tab value="Windows">
    Download and run the installer from [rustup.rs](https://rustup.rs), or use winget:
    ```powershell
    winget install Rustlang.Rustup
    ```
  </Tab>
</Tabs>

### UV (Python Environment Manager)

UV is **required** for running hooks with proper dependency management.

<Tabs items={['macOS / Linux', 'Windows']}>
  <Tab value="macOS / Linux">
    ```bash
    curl -LsSf https://astral.sh/uv/install.sh | sh
    uv --version
    ```
  </Tab>
  <Tab value="Windows">
    ```powershell
    powershell -c "irm https://astral.sh/uv/install.ps1 | iex"
    uv --version
    ```
  </Tab>
</Tabs>

<Callout type="info" title="Why UV?">
- ✅ **Cross-platform:** Works on Windows, Mac, and Linux
- ✅ **Fast:** Rust-based, 10-100x faster than pip
- ✅ **Isolated:** Manages Python environments per project
- ✅ **Zero config:** Just works out of the box
</Callout>

## Installing agentic-p

### From Source (Recommended)

<Steps>
  <Step>
    ### Clone the Repository
    ```bash
    git clone https://github.com/AgentParadise/agentic-primitives.git
    cd agentic-primitives
    ```
  </Step>
  <Step>
    ### Build the CLI
    ```bash
    # Using Just (recommended)
    just build

    # Or directly with Cargo
    cd cli && cargo build --release
    ```
  </Step>
  <Step>
    ### Install to PATH
    ```bash
    # Using Just
    just install

    # Or manually
    cp target/release/agentic-p ~/.local/bin/
    ```
  </Step>
  <Step>
    ### Verify Installation
    ```bash
    agentic-p --version
    ```
  </Step>
</Steps>

## Installing Plugins

Plugins provide commands, skills, and hooks for Claude Code agents. There are three installation methods.

### Local Install (via Just)

The fastest way to install plugins for the current project or globally:

```bash
# List available plugins with versions
just plugin-list

# Install to current project
just plugin-install sdlc
just plugin-install workspace

# Install globally (available in all projects)
just plugin-install sdlc --global

# Uninstall
just plugin-uninstall sdlc
just plugin-uninstall workspace --global

# Validate all plugin manifests
just plugin-validate
```

<Callout type="info" title="What happens on install?">
1. Plugin files are copied to `.claude/plugins/cache/agentic-primitives/{name}/local/`
2. Hooks from `hooks/hooks.json` are merged into the target `settings.json`
3. The plugin is added to `enabledPlugins`
4. Reinstalling is idempotent — no duplicate hooks
</Callout>

### Remote Install (via Claude Code Marketplace)

If the repo is pushed to GitHub, users can install plugins directly from Claude Code:

```bash
# Add the marketplace (one-time)
/plugin marketplace add AgentParadise/agentic-primitives

# Install plugins
/plugin install sdlc@agentic-primitives
/plugin install workspace@agentic-primitives
```

### Available Plugins

| Plugin | Description | Primitives |
|--------|-------------|------------|
| **sdlc** | Software Development Lifecycle | 5 commands, 7 skills, security hooks, git hooks |
| **workspace** | Observable agent workspaces | 8 lifecycle hooks |
| **meta** | Primitive generators | 3 commands |
| **research** | Information gathering | 1 command, 1 tool |
| **docs** | Documentation tools | 1 skill |

For full details on each plugin's hooks and how they compose, see the [Plugin Guide](/docs/guides/plugins).

## Post-Installation Setup

### Shell Completions

<Tabs items={['Bash', 'Zsh', 'Fish']}>
  <Tab value="Bash">
    ```bash
    agentic-p completions bash > ~/.local/share/bash-completion/completions/agentic-p
    ```
  </Tab>
  <Tab value="Zsh">
    ```bash
    agentic-p completions zsh > ~/.zfunc/_agentic-p
    # Add to ~/.zshrc: fpath+=~/.zfunc
    ```
  </Tab>
  <Tab value="Fish">
    ```bash
    agentic-p completions fish > ~/.config/fish/completions/agentic-p.fish
    ```
  </Tab>
</Tabs>

### Configuration

Create a global configuration file:

```bash
agentic-p config init --global
```

This creates `~/.config/agentic-p/config.yaml`.

## Development Setup

For contributing to agentic-p:

```bash

cd agentic-primitives

# Install git hooks

# Run the full QA suite

just build

# Run tests
just test
```

## Troubleshooting

<Accordions>
  <Accordion title="Command not found: agentic-p">
    Ensure the binary is in your PATH:
    ```bash
    # Check if installed
    which agentic-p

    # Add to PATH if needed
    export PATH="$HOME/.local/bin:$PATH"
    ```
  </Accordion>
  <Accordion title="UV not found">
    Make sure UV is installed and in your PATH:
    ```bash
    # Reinstall UV
    curl -LsSf https://astral.sh/uv/install.sh | sh

    # Check installation
    uv --version
    ```
  </Accordion>
  <Accordion title="Build errors on Windows">
    Ensure you have the Visual Studio C++ Build Tools:
    ```powershell
    winget install Microsoft.VisualStudio.2022.BuildTools
    ```
  </Accordion>
  <Accordion title="Permission denied on install">
    Use a local directory:
    ```bash
    mkdir -p ~/.local/bin
    cp target/release/agentic-p ~/.local/bin/
    export PATH="$HOME/.local/bin:$PATH"
    ```
  </Accordion>
</Accordions>
