# Python Libraries

Agentic Primitives includes production-ready Python libraries for building AI agent systems.

## Available Libraries

<Cards>
  <Card title="agentic-isolation" href="/docs/libraries/isolation">
    Workspace isolation for AI agent execution with security hardening and streaming.
  </Card>
  <Card title="agentic-events" href="#">
    Event recording and playback for testing agent workflows.
  </Card>
  <Card title="agentic-logging" href="#">
    Structured logging with correlation IDs and JSON output.
  </Card>
</Cards>

## Installation

All libraries are available via pip:

```bash
pip install agentic-isolation
pip install agentic-events
pip install agentic-logging
```

Or install from the monorepo:

```bash
cd lib/python/agentic_isolation
pip install -e .
```
