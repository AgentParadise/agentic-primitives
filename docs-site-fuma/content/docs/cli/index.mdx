## Overview

The `agentic-p` CLI provides a comprehensive set of commands for managing agentic primitives.

```bash
agentic-p [COMMAND] [OPTIONS]
```

## Global Options

| Option | Description |
|--------|-------------|
| `-h, --help` | Show help message |
| `-V, --version` | Show version |
| `-v, --verbose` | Enable verbose output |
| `-q, --quiet` | Suppress non-error output |
| `--config <PATH>` | Use custom config file |

## Quick Examples

```bash
# Initialize a new repository
agentic-p init

# Create a new agent
agentic-p new agent my-agent

# Validate all primitives
agentic-p validate --strict

# Build for Claude
agentic-p build --provider claude

# Install to project
agentic-p install --provider claude --project
```
